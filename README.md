## 🚀 Proyecto QA destacado

### 🎬 Only Film — Suite de Testing Automatizado sobre Spring Boot
> Proyecto colaborativo (4 personas) · Adecco / CertiDevs · 2026

Arquitectura MVC completa desarrollada desde cero: entidades JPA, Spring Security y suite completa de testing automatizado.

- **Unit Testing** — JUnit 5 + Mockito: servicios aislados, lógica de negocio, validaciones y disponibilidad.
- **Integration Testing** — Repository Tests (JPA/H2) y Controller Tests con MockMvc (GET, POST, redirecciones, seguridad).
- **UI Testing (Selenium)** — Flujos E2E completos: login, cartelera, selección de butaca, checkout y generación de ticket QR.
- **CI/CD** — Pipeline GitHub Actions: compilación Maven y ejecución de tests en cada push.

👉 [Ver repositorio](https://github.com/alopezdeharo/onlyfilm-testing)
